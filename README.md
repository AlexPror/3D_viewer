# 3D Viewer

Веб-приложение для просмотра 3D-моделей, чертежей (PDF) и спецификации. Совместный просмотр 2D/3D, сечение по клику, измерения с привязкой к граням/рёбрам/вершинам, экспорт в GLB/STL, скриншоты с аннотациями.

**Версия:** 1.4.0

## Возможности

### Режимы и вкладки
- **3D / 2D** — просмотр 3D-модели и PDF (чертежи).
- **Спецификация** — таблица по ГОСТ (Поз., Обозначение, Наименование, Кол., Масса ед./общ., Примечание). Данные берутся с локального сервера по загруженному STEP.

### Файлы и загрузка
- **Форматы 3D:** STL, STEP (.stp, .step), IGES (.igs, .iges), GLB/GLTF.
- **STEP/IGES:** конвертация в браузере (opencascade.js WASM) или на локальном сервере (CadQuery + trimesh). Лимит загрузки STEP/IGES: **30 МБ** (файлы больше не принимаются, в лог пишется предупреждение).
- **Drag-and-drop** — перетаскивание файла в окно (PDF — чертёж, 3D — модель).
- **Модель:** один светло-серый цвет; при необходимости — режим «Каркас» (прозрачные грани с регулируемой прозрачностью).

### Камера и вид
- **TrackballControls** — вращение (правая кнопка), панорама (средняя), zoom (колёсико).
- **Ориентация** — меню видов (Фронт, Верх, Лево, Изометрия и др.).
- **Перпендикулярно** — клик по грани модели, затем кнопка: камера разворачивается к грани. Кнопка всегда активна; без выбранной грани в лог выводится подсказка.

### Каркас (прозрачные грани)
- Отдельный блок в тулбаре и на плавающей панели сцены.
- Включение/выключение — кнопка «Каркас». Вместо проволочного вида — полупрозрачные грани (видны углы и рёбра).
- **Прозрачность (opacity)** — поле ввода и слайдер в шапке, поле ввода на панели сцены. Изменение колёсиком мыши или вводом; значение 0.05–0.95, шаг 0.05. При включённом режиме изменение сразу применяется к модели.

### Сечение
- Режим «Сечение» — клик по грани задаёт плоскость сечения по нормали. Смещение плоскости — поле, слайдер и колёсико мыши (±2000). Кнопки ✓ (зафиксировать) и ✕ (снять).

### Измерения
- Режим «Измерение» — два клика по модели, отрезок с подписью (L, ΔX, ΔY, ΔZ в мм). Привязка: точка пересечения, вершина, центр грани, ребро. Кнопка «Очистить».

### Экспорт и отчёт
- **Экспорт GLB / STL** — скачать текущую модель.
- **Скриншот 3D** — рендер сцены, редактор (фигуры, текст), сохранение в PNG или в PDF-отчёт.
- **Отчёт PDF** — шапка по первому листу PDF, скриншоты 2D/3D из панели, блок измерений.

## Стек

- **Vue 3**, **TypeScript**, **Vite**
- **Three.js** — сцена, GLTFLoader, STLLoader, экспорт GLB/STL, TrackballControls, Raycaster
- **opencascade.js** — чтение STEP/IGES в браузере, тесселяция, экспорт в GLB
- **Локальный сервер (FastAPI):** метаданные STEP (продукты, сборка), спецификация по ГОСТ, конвертация STEP→GLB (cadquery + trimesh)
- **PDF.js**, **jspdf** — просмотр PDF и формирование отчёта

## Запуск

### Всё сразу (сервер + фронт)

```batch
run.bat
```

Откроются два окна: сервер (порт 8000) и фронт (Vite). В браузере — http://localhost:5173.

### Только фронт (без спецификации и серверной конвертации STEP)

```bash
cd viewer
npm install
npm run dev
```

Откройте http://localhost:5173. Спецификация и конвертация STEP на сервере будут недоступны; STEP до 30 МБ конвертируются в браузере (WASM).

### Сервер отдельно

```powershell
cd server
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

Подробнее и API — в [server/README.md](server/README.md).

## Сборка

```bash
cd viewer
npm run build
```

Результат в `viewer/dist/`.

## Структура

- **viewer/** — фронт (Vue 3 + Vite)
  - `src/App.vue` — режимы, тулбар, отчёт
  - `src/components/Viewer3D.vue` — сцена, загрузка моделей, каркас, сечение, измерения, спецификация
  - `src/lib/stepLoader.ts` — STEP/IGES → GLB (opencascade.js)
- **server/** — FastAPI: метаданные STEP, спецификация, конвертация STEP→GLB
- **spec_src/** — исходники логики спецификации (материалы, секции)
