# 3D Viewer

Веб-приложение для просмотра 3D-моделей и PDF: совместный просмотр чертежей и модели, сечение по клику, измерения с привязкой к граням/рёбрам/вершинам, экспорт в GLB/STL, скриншоты с аннотациями.

**Версия:** 1.1.0

## Возможности

### Режимы просмотра
- **2D** — только PDF (чертежи, спецификация).
- **3D** — только 3D-модель.
- **Совместный просмотр** — две панели 50/50 (PDF слева, модель справа).

### Файлы и камера
- **PDF:** кнопка «Открыть 2D PDF» — выбор файла, просмотр через PDF.js, переключение страниц.
- **3D-модель:** кнопка «Открыть 3D модель» — поддержка форматов **STL**, **STEP** (.stp, .step), **IGES** (.igs, .iges), **GLB/GLTF**. STEP/IGES конвертируются в браузере через [opencascade.js](https://github.com/OpenCascade.js/opencascade.js) (WASM) в GLB.
- **Drag-and-drop** — перетаскивание файла в окно приложения: PDF открывается как чертёж, 3D-файл (STL, STEP, IGES, GLB) — как модель. При перетаскивании показывается подсказка.
- **Вид по умолчанию** — сброс камеры к сохранённому положению после загрузки модели.
- **Управление камерой:** TrackballControls — вращение (правая кнопка мыши), панорама (средняя), плавный zoom (колёсико).

### Сечение
- **Сечение** — режим «клик по модели»: левый клик по грани задаёт плоскость сечения по нормали грани (как в CAD). Отображается полупрозрачная плоскость разреза.
- **Смещение плоскости:** при активном сечении в тулбаре доступны поле ввода и слайдер; значение можно менять колёсиком мыши. Диапазон ±2000.
- **Зафиксировать сечение (✓)** — выйти из режима выбора плоскости, оставив текущее сечение. После фиксации можно делать скриншот и проводить измерения.
- **Снять сечение (✕)** — убрать плоскость и отсечение.

### Измерения
- **Измерение** — режим «два клика по модели»: строится отрезок между точками, подпись с длиной **L** и приращениями **ΔX, ΔY, ΔZ** (в мм).
- **Привязка к геометрии** (выпадающий список):
  - **Точка пересечения** — точка пересечения луча с гранью.
  - **Вершина** — ближайшая вершина грани.
  - **Центр грани** — центр треугольника грани.
  - **Ребро** — ближайшая точка на ребре грани.
- **Очистить измерения** — удалить отрезок и подпись.

### Экспорт и скриншот
- **Экспорт GLB** — скачать текущую 3D-модель в формате GLB (бинарный glTF).
- **Экспорт STL** — скачать текущую модель в формате STL (бинарный).
- **Отчёт PDF** — формирование и скачивание PDF-отчёта: скриншот текущей страницы PDF (если открыт), скриншот 3D-модели, блок «Измерения» (длина и ΔX, ΔY, ΔZ в мм). Имя файла: `Отчет_<имя>_<дата>.pdf`.
- **Скриншот** — выбор: «Скриншот 2D (PDF)» (текущая страница) или «Скриншот 3D модели»; открывается редактор скриншота (фигуры, текст, обрезка), сохранение в PNG.

### Тулбар
- Кнопки сгруппированы по назначению с разными цветами:
  - **Файл и вид** — PDF, 3D модель, вид по умолчанию, переключатель 2D/3D/Совм.
  - **Сечение** — Сечение, ✓ (фиксация), ✕ (снять), смещение (при активном сечении).
  - **Измерение** — Измерение, Очистить, выбор привязки.
  - **Экспорт** — Скриншот, Экспорт GLB, Экспорт STL, Отчёт PDF.

## Стек

- **Vue 3**, **TypeScript**, **Vite**
- **Three.js** — сцена, камера, GLTFLoader, STLLoader, GLTFExporter, STLExporter, TrackballControls, Raycaster
- **opencascade.js** — чтение STEP/IGES, тесселяция, экспорт в GLB
- **PDF.js** — просмотр PDF
- **jspdf** — формирование PDF-отчёта (скриншоты + измерения)

## Запуск

```bash
cd viewer
npm install
npm run dev
```

В браузере откройте адрес, который выведет Vite (например http://localhost:5173).

## Сборка

```bash
cd viewer
npm run build
```

Результат в `viewer/dist/`.

## Версионность

Версия проекта задаётся в `viewer/package.json` (поле `version`). Формат — [SemVer](https://semver.org/) (major.minor.patch).

## Структура

- **viewer/** — приложение (Vue 3 + Vite)
  - **viewer/src/App.vue** — корневой компонент, состояние режимов, обработчики тулбара
  - **viewer/src/components/Viewer3D.vue** — сцена Three.js, загрузка моделей, сечение, измерения, экспорт
  - **viewer/src/components/ViewerToolbar.vue** — тулбар с группами кнопок
  - **viewer/src/components/PdfViewer.vue** — просмотр PDF
  - **viewer/src/components/ScreenshotEditorModal.vue** — редактор скриншота
  - **viewer/src/lib/stepLoader.ts** — загрузка STEP/IGES через opencascade.js
- **docs/** — заметки, анализ форматов (например JT-analysis.md)
