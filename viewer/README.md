# 3D Viewer — приложение

Вьюер 3D-моделей: STEP, STL, IGES. Часть проекта [3D_viewer](../README.md).

**Версия:** см. `package.json` (поле `version`).

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес от Vite (обычно http://localhost:5173).

## Реализовано

- **Сцена Three.js:** камера, освещение, TrackballControls (вращение — правая кнопка, панорама — средняя, zoom — колёсико).
- **Загрузка STL:** выбор .stl → модель в сцене, авто-центрирование.
- **Загрузка STEP / IGES:** конвертация в браузере через opencascade.js (WASM) → GLB → отображение. Грубая тесселяция (0.5) для быстрой загрузки.
- **Скриншот:** кнопка «Скриншот» → модальное окно с изображением, редактор (стрелка, прямоугольник, эллипс, текст, обрезка), сохранение PNG.
- **Вид по умолчанию:** кнопка сброса камеры к виду после загрузки модели.
- **Таймер загрузки:** в консоли выводятся этапы (WASM, чтение, тесселяция, экспорт). Предзагрузка WASM при старте.
