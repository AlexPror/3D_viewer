# 3D Viewer — STEP, STL, IGES, DXF

Вьюер для проектов КОМПАС-3D (по аналогии с eDrawings): просмотр моделей, скриншот с аннотациями, разнесение сборки.

## Запуск

```bash
cd viewer
npm install
npm run dev
```

Откройте в браузере адрес, который выведет Vite (обычно http://localhost:5173).

## Реализовано

- **Сцена Three.js**: камера, освещение, OrbitControls (приближение, обзор 360°).
- **Загрузка STL**: кнопка «Открыть файл», выбор .stl → модель в сцене, авто-центрирование.
- **Скриншот**: кнопка «Скриншот» → модальное окно с изображением текущего вида.
- **Редактор скриншота**: стрелка, прямоугольник, эллипс, обрезка (crop), «Сохранить на ПК», «Отправить в чат» (заглушка).

## В планах

- STEP, IGES, DXF через opencascade.js.
- Разнесение компонентов сборки (ползунок как в eDrawings).

## Тестовые файлы

Образцы STEP/STL лежат в `../project_m1/` (например, `Сборка модуля М-1-1.1(К3).stp`).
